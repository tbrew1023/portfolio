JQuery Puzzle

Find all the missing items without errors for a high score!

Project Summary:

This project was a puzzle game I made for my Interactive Web Scripting course at DePaul. I used the JQuery library along with HTML and CSS to build this puzzle.

Creative Process:

JQuery was perfect for this project, considering it has a lot of visual effects that would be difficult to implement using raw JavaScript. Jquery was used to expand containers, change backgorund colors, and dynamically change certain counters in the UI.

Results:

This project was a lot of fun and I find myself coming back to play it every now and again. My favorite part is when you finally win and you get to see what grade you got along side a little message that roasts you if you did bad:)
